# 項目內容

參考SpaceRat進行程序開發，於ESP32實現BLE HID為主要功能，
搭配EC12旋轉編碼器與MPU6500進行滑鼠控制。

### 準備材料

- ESP32
- MPU6500 (MPU6050)
- LCD Moudle SSD1308 I2C 0.96
- EC12/11
